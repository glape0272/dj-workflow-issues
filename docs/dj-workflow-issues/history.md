# プロジェクト履歴

## 2026-05

- 初期環境: DJ ワークフロー課題を整理するための README、課題分類ディレクトリ、USB 管理カテゴリ、記録テンプレートを作成
- Mixxx課題整理: ライブラリ依存、検索性、性能、コントローラー対応、初期導入、解析精度に関する課題メモを追加し、`problems/` 配下を自分の問題点と他者由来の問題点に分けて整理
- 運用整理: 問題メモを `personal/` と `external/` で分け、カテゴリ分類、出典記録、関連ファイル同時更新の方針を README、ディレクトリ運用メモ、agent 用メモへ反映
- Mixxx外部情報整理: 公式リポジトリ、公式 Wiki、利用者投稿から、個人製作の外部アプリで補助できそうな Mixxx の外部由来課題を `external/` 配下へ追加し、ソースに確認元リンクを記録する方針を整理
- テンプレート整理: 現在の運用に合わせて、問題メモテンプレートへ出どころ、ソース、外部アプリで対応できそうな範囲の項目を追加
