# Mixxx 問題メモ

Mixxx 固有の問題点を出どころとカテゴリごとに整理する場所

## 出どころ

- [自分が感じている問題点](./personal/)
- [他の人が感じている問題点](./external/)

## カテゴリ

自分が感じている問題点:

- [ライブラリ](./personal/library/)

- [初期導入](./personal/setup/)

他の人が感じている問題点:

- [解析](./external/analysis/)

## 記録方針

- 1つの問題につき1つの Markdown ファイル
- ファイル名は `YYYY-MM-DD-short-topic.md` を基本形
- `templates/issue-note.md` をコピーして作成
- 関連する問題メモがある場合は相互リンクを追加
- 自分の問題点は `personal/` 配下へ配置
- 他の人の問題点は `external/` 配下へ配置
- `external/` では、個人製作の外部アプリで補助・変換・検証・可視化できそうな問題を優先
- 外部情報を根拠にする場合は `ソース` に確認元リンクを記録
- カテゴリが増えた場合はこの README と Mixxx 側 README も更新
