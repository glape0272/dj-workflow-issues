# 対応コントローラーが少ない

## 概要

- 対象ツール: Mixxx
- 発生日: 2026-05-07
- 優先度: 低
- 状態: 未調査

## 困っていること

公式サポート外のコントローラーは、自分で JavaScript などを書いてマッピングする必要がある

Mixxx の自由度の高さでもあるが、初心者や機材を頻繁に変えるユーザーには負担になる

## 再現条件

- 公式サポート外のDJコントローラーを利用
- MIDI/HIDマッピングが未提供
- コントローラー固有の動作調整が必要

## 期待する状態

- 使用したいコントローラーを短時間で利用開始できる状態
- マッピング作成や調整の負荷が小さい状態
- 対応可否や必要作業を事前に判断できる状態

## 現時点の回避策

- 公式サポート済みコントローラーを選ぶ運用
- 既存のコミュニティマッピングを探す運用
- JavaScript などで自作マッピングを作成

## 調査メモ

- DB構造そのものとの関係は薄い
- ライブラリ操作やCue操作をコントローラー側から扱う場合は、内部データ構造の理解が補助になる可能性
