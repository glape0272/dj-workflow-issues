# Serato 問題メモ

Serato 固有の問題点を出どころとカテゴリごとに整理する場所

## 出どころ

- [自分が感じている問題点](./personal/)

## カテゴリ

自分が感じている問題点:

- [ライブラリ](./personal/library/)

## 記録方針

- 1つの問題につき1つの Markdown ファイル
- ファイル名は `YYYY-MM-DD-short-topic.md`
- `templates/issue-note.md` をコピーして作成
- 関連する問題メモがある場合は相互リンクを追加
- 自分の問題点は `personal/` 配下へ配置
- 他の人の問題点は `external/` 側へ配置
- カテゴリが増えた場合はこの README と Serato 側 README も更新
