# Cross Tool 問題メモ

Mixxx、Serato、Spotify、USB にまたがる問題点を出どころとカテゴリごとに整理する場所

## 出どころ

- [自分が感じている問題点](./personal/)

## カテゴリ

自分が感じている問題点:

- [プレイリスト](./personal/playlist/)
- [ライブラリ](./personal/library/)
- [メタデータ](./personal/metadata/)
- [自動化](./personal/automation/)
- [ワークフロー](./personal/workflow/)

## 記録方針

- 1つの問題につき1つの Markdown ファイル
- ファイル名は `YYYY-MM-DD-short-topic.md`
- `templates/issue-note.md` をコピーして作成
- 関連するツール固有問題がある場合は相互リンクを追加
- 複数ツールにまたがる問題のみ配置
- ツール固有の制約は各ツール側の問題メモへ配置
