# 自分が感じている横断問題

自分の DJ ワークフローで実際に困っている、複数ツールにまたがる問題点

## 記録方針

- 1つの問題につき1つの Markdown ファイル
- 自分の体験、運用、検証結果にもとづく問題点を記録
- ファイル名は `YYYY-MM-DD-short-topic.md` を基本形
- 必要に応じて `playlist/`、`library/` などのカテゴリへ配置
- ツール固有問題と重なる場合は、横断側には連携で発生する困りごとを記録
