# ファイル名とタグ情報が一致しない

## 概要

- 対象ツール: USB / Mixxx / Serato
- 発生日: 2026-05-08
- 優先度: 中
- 状態: 未調査
- 出どころ: personal

## 困っていること

MP3タグが未整備、またはファイル名とタグ情報が一致していないと、DJソフト側の表示、検索、並び替えが不安定になりやすい

複数のディレクトリやUSBで同じ曲を扱う場合、どの情報を正として修正すべきか判断しにくい状態

## 再現条件

- MP3タグが未整備
- ファイル名と曲名、アーティスト名、アルバム名が一致していない
- 日本語、特殊記号、文字コードの差異が混在
- Mixxx と Serato の両方で同じ音源を利用

## 期待する状態

- MP3タグの必須項目と表記ルールが決まっている状態
- ファイル名とタグ情報の差異を確認できる状態
- DJソフト側の表示や検索で迷いにくい状態

## 現時点の回避策

- MP3Tag などで曲名、アーティスト名、アルバム名などのタグを整える
- ファイル名とタグ情報のどちらを正とするかを決める
- 日本語や特殊記号の扱いを統一する

## 個人製作アプリで対応できそうな範囲

- ファイル名とタグ情報の不一致検出
- タグ必須項目の欠落検出
- 表記ゆれ候補の一覧化
- 日本語、特殊記号、文字コード、パス長のチェック

## 対応が難しそうな範囲

- タグ情報が存在しない音源の正確な自動補完
- 正しい曲名、アーティスト名の完全自動判定
- DJソフトごとの表示差の完全吸収

## 調査メモ

- ファイル名だけを正とするとDJソフト側の表示とずれる可能性
- タグだけを正とするとファイル単体で見たときに判別しにくい可能性
- 日本語、特殊記号、文字コード、パス長の問題も確認が必要
