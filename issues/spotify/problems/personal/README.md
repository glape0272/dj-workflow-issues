# 自分が感じている Spotify の問題点

自分の DJ ワークフローで実際に困っている Spotify 固有の問題点

## 記録方針

- 1つの問題につき1つの Markdown ファイル
- 自分の体験、運用、検証結果にもとづく問題点を記録
- ファイル名は `YYYY-MM-DD-short-topic.md` を基本形
- 必要に応じて `library/`、`playlist/` などのカテゴリへ配置
- 外部情報を参照した場合は `ソース` または `関連リンク` に確認元を記録
- 他の人の意見を主な根拠にする場合は `external/` 側へ配置
