# Spotify APIでプレイリスト操作を補助したい

## 概要

- 対象ツール: Spotify
- 発生日: 2026-05-08
- 優先度: 中
- 状態: 未調査
- 出どころ: personal

## 困っていること

Spotify のお気に入りの曲やプレイリスト周りを、外部アプリからAPIなどで操作できると整理が楽になりそう

特にリスト作成、未分類曲の抽出、分類候補の生成、プレイリストへの追加を補助できれば、大量曲の整理負荷を下げられる可能性

## 再現条件

- お気に入りの曲が大量にある
- 複数のSpotifyプレイリストへ曲を振り分けたい
- 手動操作だけでは整理に時間がかかる
- 外部アプリからSpotify APIを使った操作を検討している

## 期待する状態

- 外部アプリからお気に入りの曲やプレイリストを参照できる状態
- 未分類曲や分類候補を一覧化できる状態
- 確認後にプレイリストへ追加できる状態
- 誤分類や誤操作を避けるため、変更前に確認できる状態

## 現時点の回避策

- Spotify本体で手動整理する
- プレイリストごとに曲を検索して追加する
- お気に入りの曲を追加日順に確認する

## 個人製作アプリで対応できそうな範囲

- Spotify API を使ったお気に入りの曲の取得
- プレイリスト一覧と所属曲の取得
- 未分類曲、重複所属曲、候補リストの生成
- 確認済み候補のプレイリスト追加
- 整理状況のダッシュボード化

## 対応が難しそうな範囲

- Spotify API の権限や制限を超える操作
- Spotify側で提供されていない内部情報の取得
- 誤分類を完全に防ぐ自動プレイリスト編集
- Serato側の検索UIや参照仕様の改善

## 調査メモ

- まずSpotify APIで取得、追加、削除、更新できる範囲の確認が必要
- 自動変更よりも、候補提示と確認後反映のほうが安全
- プレイリスト周りが外部アプリの主な対応範囲になりそう
